## UTK RASC-AL Competition

###Run:
```
sudo bash
cd /home/pi/
apt-get update
git clone https://github.com/gecostigan/rascal17
pip install -r requirements.txt
sudo apt-get install xboxdrv
chmod 755 start.sh
./start.sh
```
